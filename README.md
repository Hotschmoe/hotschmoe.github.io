# hotschmoe.github.io

Personal portfolio site built with Jekyll and GitHub Pages.

## Local Development

```bash
# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve

# View at http://localhost:4000
```

## Adding Projects

1. Copy your built project folder to the root directory
2. Add a link to it in `index.md`
3. Commit and push

## Custom Font Setup

See `assets/css/style.scss` for font configuration.
